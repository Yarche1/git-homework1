Ответы на тест по ООП:
1. Класс - шаблон для объектов.
2. Объект - экземпляр класса.
3. Конструктор - метод для инициализации объекта.
4. private - закрывает доступ к полям.
5. public - открывает доступ к полям/методам.
6. Наследование - создание нового класса на базе старого.
7. class Warrior : public Character - наследование Warrior от Character.
8. Геттеры - безопасное чтение private полей.
9. -> - доступ к члену класса через указатель.
10. Инкапсуляция - защита и сокрытие данных.
11. new - выделение динамической памяти.
